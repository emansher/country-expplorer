# Fetch API Project

## Description
This project demonstrates the use of the JavaScript Fetch API to retrieve and display user data from a public API.

## Features
- Fetches live data from DummyJSON API
- Search users by name or country
- Loading state
- Error handling
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- Fetch API
